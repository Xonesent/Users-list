typical repository structure

|
|- _materials - материалы
|-- Images - картинки
|
|- cmd - запускаемый файл
|- configs - конфиги
|
|- pkg - функционал сервиса
|-- handler - слой с хэндлерами и их реализацией
|--- api - сами функции
|--- helpers - валидация входных данных
|
|-- repository - слой работы с базой данных
|--- api - сами функции
|--- helpers - конструктор sql запросов
|
|-- service - слой бизнес логики
|--- api - сами функции
|--- helpers - обогащатель get метода
|
|- schema - миграции
|- server - функции по работе с сервером
|- .env - (должен быть скрыт, но скрывать особо и нечего)
|- go.mod
|- go.sum
|- README.md - о репозитории + впечатления
